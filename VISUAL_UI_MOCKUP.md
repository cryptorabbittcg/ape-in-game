## 💡 ANIMATION SEQUENCES

### Page Load:
```
1. Particles fade in (0s)
   → Gentle appearance

2. Banner scales up (0.5s)
   → Bouncy spring animation

3. Title fades + slides up (0.7s)
   → With glow pulse

4. Cards stagger in (0.8s - 1.6s)
   → Each 0.1s apart
   → Scale from 80% to 100%

5. Instructions fade in (1.8s)
   → Smooth opacity transition
```

### Card Hover:
```
Mouse enters card
   ↓
Icon bounces (0.2s)
   ↓
Card lifts Z-axis (0.3s)
   ↓
Glow intensifies (0.3s)
   ↓
Background pattern animates
   ↓
3D tilt based on mouse position
   ↓
Mouse leaves
   ↓
Everything smoothly returns
```
